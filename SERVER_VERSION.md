# L2Jaln Server Version Documentation

Esta documentação detalha a versão do servidor L2Jaln, baseado no aCis 382 (High Five) e com **total compatibilidade com clientes Interlude (Protocolo 730) e High Five (Protocolo 746)**.

## Versão do Servidor
- **Base:** aCis 382 (Int/HF)
- **Protocolo:** 730 (Interlude)
- **Protocolo:** 746 (High Five)
- **Login Server Protocol:** 258
- **Game Server Protocol:** 258 (0x0102)

## Compatibilidade com Clientes
- **Protocolo Mínimo:** 730 (Interlude)
- **Protocolo Máximo:** 746 (High Five)
- **Clientes Suportados:**
  * Interlude (730)
  * Interlude+ (740-745)
  * High Five (746)

## Diferenças entre aCis 372 e aCis 382

### aCis 372 (Interlude)
- Protocolo 730
- Mais leve
- Menos recursos
- Mais estável
- Mais fácil de configurar
- Melhor para servidores pequenos
- Menos bugs
- Mais compatível com mods antigos

### aCis 382 (High Five) - Nossa Versão
- Protocolo 746
- Mais recursos
- Sistemas mais avançados
- Melhor performance
- Melhor segurança
- Melhor estabilidade
- Melhor para servidores grandes
- Mais difícil de configurar
- Requer mais recursos do servidor

## Recomendações para Uso

### Cliente
1. Use um cliente High Five limpo
2. Configure o cliente para usar UTF-8
3. Verifique compatibilidade com todos os sistemas
4. Teste todos os sistemas importantes

### Servidor
1. Monitore o desempenho do servidor
2. Mantenha backups regulares
3. Documente todas as alterações
4. Mantenha o sistema atualizado
5. Verifique logs regularmente

### Segurança
1. Mantenha o HWID ativado
2. Configure proteção contra flood
3. Mantenha o sistema de logging ativo
4. Faça backup regular do banco de dados
5. Monitore tentativas de invasão
